# DSA A-to-Z Cheatsheets

Welcome! These files combine verbatim code from your repo with quick revision notes.

Legend
- 🗺️ Quick map: skim the section list and navigate.
- ✅ Study checklist: sanity checks before closing a topic.
- 🧠 Notes: patterns, pitfalls, and decisions.
- 🧩 Snippets: verbatim code you’ve already written.

Tips
- Skim Quick map first, then the checklist, then snippets.
- When time-crunched, review the checklists only; drill into snippets you’re weak on.
- Prefer your own code as ground truth; notes only add context.

Quick-reference notes per topic, compiled from this repo’s READMEs and patterns. Use them for fast revision.

Topics covered
- Arrays, Binary Search (+Refresher), BSTs, Binary Trees
- Bit Manipulation, Fenwick Trees, Segment Trees
- Graphs, Trees (LCA/Binary Lifting), Tries
- DP (incl. DP on Trees), Prefix/Partial Sums
- Strings (KMP, Rolling Hash), Sweep Lines
- Two Pointers/Sliding Window, Stacks/Queues, Heaps, Greedy
- LinkedList, Math, Recursion/Backtracking
- CSES, Codeforces Practice, LC Contests, Others

Each sheet is intentionally concise and links to deeper notes where applicable.
